# 🎭 Декорации тайлов Generative Map

## Принципы декораций

- **Стиль**: Минималистичный пиксель-арт / векторные иконки
- **Размер**: Читаемы при размере тайла 80-120px
- **Количество**: 1-3 декорации на тайл (не перегружать!)
- **Позиционирование**: Распределены по тайлу, не центрированы

---

## 🌊 Вода / Океан

### Декорации
1. **Волны** — три параллельные линии, изогнутые
2. **Пена** — белые точки по краям
3. **Рыбка** — простая иконка рыбы (редкая)
4. **Коралл** — разветвлённая структура (редкая)

### Визуальное описание
```
Волны:
  ~~~  →  три волнистые линии

Пена:
  •  •   →  точки по краям тайла
    •
```

### Рельеф
- Плоский (спокойная вода)
- Волнистый (небольшие волны)
- Бурный (большие волны + пена)

## 🌲 Лес

### Декорации
1. **Деревья** — 3 варианта формы
   - Круглая крона (дуб)
   - Треугольная крона (ель)
   - Овальная крона (берёза)
2. **Кусты** — круглые зелёные пятна
3. **Грибы** — маленькие точки с шляпками
4. **Цветы** — точки разных цветов

### Визуальное описание
```
Дерево (дуб):     Дерево (ель):     Куст:
    🌳              🎄              🌿
   /||\            /|\            (  )
    ||             |              

Вариации размера: маленькое / среднее / большое
```

### Расположение
- 1 большое дерево в центре
- 2-3 средних дерева
- 4-5 кустов

## 🏔️ Гора / Скалы

### Декорации
1. **Пики** — треугольники разной высоты
   - Острый пик
   - Скруглённый пик
   - Двуглавый пик
2. **Скальные образования** — серые прямоугольники
3. **Снег на вершине** — белая полоска сверху
4. **Водопад** — тонкая синяя линия (редко)

### Визуальное описание
```
Острый пик:       Двуглавый:        Со снегом:
    /\            /\/\             /\⚪
   /  \          /    \           /  \
  /    \        /      \         /    \

Высоты: низкая / средняя / высокая
```

### Рельеф
- Плоское плато
- Холмистый склон
- Крутой пик

## 🏜️ Пустыня

### Декорации
1. **Кактус** — зелёный овал с вертикальными линиями
2. **Песчаная дюна** — изогнутые линии
3. **Камни / валуны** — коричневые овалы
4. **Череп животного** — белый круг (редкий, весёлый)
5. **Оазис** — синий круг с пальмой (супер редкий)

### Визуальное описание
```
Кактус:           Дюны:            Камень:
   🌵             ~~~~            [    ]
  /||\          (песчаные          /    \
   ||            волны)           /______\

Вариации кактуса: 1-2-3 сегмента
```

## 🌾 Равнина / Луга

### Декорации
1. **Трава** — вертикальные линии разной высоты
2. **Цветы** — точки цветов: жёлтые, белые, фиолетовые
3. **Камни** — серые овалы
4. **Пшеничные колосья** — жёлтые вертикальные линии
5. **Домик** — маленький дом (редко, поселение)
6. **Дорога** — коричневая извилистая линия

### Визуальное описание
```
Трава:            Цветы:           Домик:
 | | |           •  *  •            /\
 | ||            жёлтый            /  \
||| |            белый           |🏠|
                  фиолетовый      |__|
```

### Вариации
- Цветущий луг (много цветов)
- Ферма (пшеница + домик)
- Пастбище (трава + камни)

### Декорации

3. **Иглу** — белый полукруг

## 🎯 Алгоритм генерации декораций

```python
def generate_decorations(username: str, biome: str) -> list:
    """
    Генерирует уникальные декорации на основе хеша имени
    """
    hash_value = hash(username) % 10000
    decorations = []
    
    # Количество декораций (1-3)
    count = (hash_value % 3) + 1
    
    # Выбор типов декораций
    available = BIOME_DECORATIONS[biome]
    for i in range(count):
        dec_hash = (hash_value // (10 ** i)) % len(available)
        decoration = available[dec_hash]
        
        # Позиция на тайле
        x = (hash_value + i * 137) % 80 + 10  # 10-90%
        y = (hash_value + i * 239) % 80 + 10  # 10-90%
        
        # Размер
        size = ['small', 'medium', 'large'][(hash_value + i) % 3]
        
        decorations.append({
            'type': decoration,
            'x': x,
            'y': y,
            'size': size
        })
    
    return decorations
```

## 🎨 Стиль отрисовки

### Рекомендуемый стиль
- **Пиксель-арт**: 16×16 или 32×32 px на декорацию
- **Цвета**: Flat colors без градиентов
- **Обводка**: 1-2px чёрная или тёмная обводка
- **Тени**: Простая тень 2-4px вниз-вправо
