# 🎬 Анимация появления тайла Generative Map

## Концепция анимации

**Эффект**: "Вырастание" тайла из центра с эффектом масштабирования и появления декораций.

**Длительность**: 3-4 секунды (оптимально для Reels)

**Стиль**: Плавный, органичный, с элементами "магического" появления

---

## 🎞️ Технические параметры анимации

### Easing functions

| Фаза | Easing | Описание |
|------|--------|----------|
| Появление формы | `ease-out-back` | Overshoot для "вырастания" |
| Рост декораций | `ease-out-elastic` | Пружинный эффект |
| Стабилизация | `ease-out` | Плавное затухание |
| Свечение | `ease-in-out` | Симметричное пульсирование |

### Тайминг

```
0.0s ────── 0.5s ────── 1.0s ────── 1.5s ────── 2.0s ────── 2.5s ────── 3.0s
│            │           │           │           │           │           │
│  Пустой    │  Частицы  │   Рост    │  Цвет+    │Декорации  │ Свечение  │  Готово
│   тайл     │   центр   │   формы   │  текстура │           │           │
```

### Длительности по фазам

| Фаза | Длительность | Easing |
|------|--------------|--------|
| Частицы | 0.5s | ease-out |
| Рост формы | 0.5s | ease-out-back |
| Цвет/текстура | 0.5s | ease-out |
| Декорации (все) | 0.5s | ease-out-elastic |
| Свечение | 0.5s | ease-in-out |

## 🎨 Варианты анимации по биому

### 🌊 Вода
- Частицы: голубые/белые пузырьки
- Рост: "наполнение" жидкостью снизу вверх
- Декорации: волны "накатываются" с краёв

### 🌲 Лес
- Частицы: зелёные листочки
- Рост: "прорастание" из земли
- Декорации: деревья "растут" снизу вверх

### 🏔️ Гора
- Частицы: серые камешки/пыль
- Рост: "выпирание" из земли
- Декорации: пики "возвышаются"

### 🏜️ Пустыня
- Частицы: песчинки (жёлтые)
- Рост: "насыпание" песка
- Декорации: кактус "растёт" медленно

### 🌾 Равнина
- Частицы: травяные семена
- Рост: "заростание" травой
- Декорации: цветы распускаются

### ❄️ Снег
- Частицы: снежинки
- Рост: "замерзание" из центра
- Декорации: ёлка покрывается снегом

## 📱 Рекомендации для Reels

### Оптимизация под формат
- **Длительность**: 3-4 секунды идеально
- **Фокус**: Карта центрирована, тайл крупно
- **Повтор**: Можно зациклить для нескольких подписчиков

### Хуки для внимания
1. **Начало**: "Новый подписчик!" + звук уведомления
2. **Середина**: Показать имя + процесс генерации
3. **Конец**: "Спасибо!" + призыв подписаться

### Звуковое сопровождение (опционально)
- 0.0s — звук уведомления (ping)
- 1.0s — звук "роста" (whoosh/organic)
- 2.5s — звук "магии" (chime)
- 3.0s — финальный звук (ding)
