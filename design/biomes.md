## 🎯 Алгоритм генерации цвета из имени

```python
def generate_tile_color(username: str, biome: str) -> str:
    """
    Генерирует уникальный цвет для тайла на основе:
    - username (хеш имени)
    - biome (тип биома)
    """
    # Хеш имени → hue offset в пределах ±15° от базового цвета биома
    hash_value = hash(username) % 1000
    
    # Базовый HSL для биома
    base_hsl = BIOMES[biome]['base_hsl']
    
    # Вариации
    hue_variation = (hash_value % 30) - 15  # ±15°
    sat_variation = (hash_value % 20) - 10   # ±10%
    light_variation = (hash_value % 16) - 8   # ±8%
    
    return HSL(
        h=base_hsl.h + hue_variation,
        s=clamp(base_hsl.s + sat_variation, 20, 80),
        l=clamp(base_hsl.l + light_variation, 30, 70)
    )
```

### Пример
- Имя "Alex" → лесной тайл с оттенком #4A8F5C
- Имя "Maria" → лесной тайл с оттенком #6BAB7A

## 🎨 Визуальная гармония

### Контрасты
- Океан ↔ Пустыня: сильный холодный/тёплый контраст
- Лес ↔ Снег: естественный сезонный контраст
- Гора ↔ Равнина: текстурный контраст

### Соседство
- Вода должна касаться любого биома
- Пустыня → Равнина → Лес (плавный градиент)
- Снег только на горных тайлах или северных краях
