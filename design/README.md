# 🗺️ Generative Map — Визуальная концепция

## Описание проекта
Автоматическая генерация абстрактной карты для Instagram Reels, где каждый подписчик = уникальный тайл (участок земли). Имя подписчика определяет цвет, рельеф и декорации тайла.

## Файлы концепции

- [biomes.md](./biomes.md) — Палитра биомов (6 типов тайлов)
- [decorations.md](./decorations.md) — Декорации тайлов
- [animation.md](./animation.md) — Storyboard анимации появления
- [frame-layout.md](./frame-layout.md) — UI/UX фрейм Reels

## Визуальный стиль

### Общая эстетика
- **Стиль**: Минималистичный, абстрактный, читаемый при малом размере
- **Вдохновение**: Civilization mini-maps, настольные игры (Catan, Risk), пиксель-арт карты
- **Форма тайлов**: Шестиугольники (hex grid) — классика для карт
- **Цветовая система**: Гармоничная палитра с чёткой идентификацией биомов

### Принципы дизайна
1. Читаемость при размере тайла ~80-120px
2. Уникальность каждого тайла через цвет + декорации
3. Плавные переходы между соседними биомами
4. Визуальная иерархия: карта центральная, UI минималистичный

## Генерация от имени
Имя подписчика хешируется в:
- **Цвет тайла** — HSL из диапазона биома
- **Тип рельефа** — плоский / холмистый / горный
- **Декорации** — деревья / скалы / дома / флаги

## Технические параметры
- Формат: Instagram Reels 1080×1920
- FPS: 30
- Длительность: 3-5 секунд на тайл
- Цветовая модель: sRGB
