# 🎨 Референсы стилизованных карт Generative Map

## Собрано: 10 ключевых референсов

---

## 1. 🎮 Civilization VI — Mini Map

**Ссылка**: https://civilization.fandom.com/wiki/Map

**Описание**:
- Шестиугольная сетка (hex grid)
- 6 типов биомов: океан, равнина, лес, пустыня, тундра, снег
- Минималистичные иконки городов и юнитов
- Пиксель-арт текстуры тайлов

**Что подходит**:
- ✅ Система шестиугольников — идеально для проекта
- ✅ Цветовая дифференциация биомов
- ✅ Читаемость при малом размере
- ⚠️ Можно упростить текстуры (убрать шум)

---

## 2. 🎲 Catan (Колонизаторы) — Игровое поле

**Ссылка**: https://www.catan.com/

**Описание**:
- Классические шестиугольные тайлы
- 5 типов ресурсов: лес, глина, поле, луга, руда
- Плоские цвета без текстур
- Минималистичный дизайн

**Что подходит**:
- ✅ Идеальная простота форм
- ✅ Плоские цвета — наш стиль
- ✅ Отсутствие лишних деталей
- ✅ Гармоничная цветовая палитра

## 🎯 Рекомендации для проекта

### Основные влияния:

1. **Catan** → Плоские цвета, простота форм
2. **Dorfromantik** → Генеративная концепция, пастельные тона
3. **Civilization VI** → Hex grid система, иконки
4. **Mini Motorways** → Абстрактный минимализм

### Итоговый стиль:
- **Форма**: Шестиугольники (hex grid)
- **Цвета**: Плоские, пастельные, гармоничные
- **Декорации**: Пиксель-арт, минималистичные
- **UI**: Чистый, без лишних элементов
- **Анимация**: Плавная, "органичная"
