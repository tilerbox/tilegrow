## Рекомендуемый вариант: A (квадратная карта)

**Почему**:
- Тайлы видны крупно → лучше видны детали
- Место для текста снизу
- Безопасная зона Instagram не перекрывает важное
- Профессиональный вид (как мини-игра)

- Карта должна быть видна полностью

## Анимация UI

### Появление текста

```
Кадр 1 (0s):     Кадр 2 (0.3s):    Кадр 3 (0.5s):
┌─────────┐     ┌─────────┐       ┌─────────────┐
│         │     │         │       │ ╔═════════╗ │
│  [карта]│     │  [карта]│       │ ║ @user   ║ │
│         │     │         │       │ ╚═════════╝ │
└─────────┘     └─────────┘       └─────────────┘
                                    ↑ slide up + fade
```

### Параметры анимации UI

| Элемент | Анимация | Длительность | Delay |
|---------|----------|--------------|-------|
| Фон overlay | Fade in | 0.3s | 2.0s |
| Имя | Slide up + fade | 0.4s | 2.1s |
| Тип тайла | Fade in | 0.3s | 2.3s |
| Спасибо | Scale + fade | 0.3s | 2.6s |
| CTA | Slide up | 0.4s | 2.9s |

## Технические заметки для разработки

### Размеры в пикселях

```python
REELS_WIDTH = 1080
REELS_HEIGHT = 1920

# Карта (квадрат, центрирована)
MAP_SIZE = 1080
MAP_X = 0
MAP_Y = (1920 - 1080) // 2 - 100  # ~420px сверху

# UI зоны
TOP_SAFE = 100      # Статус-бар
BOTTOM_SAFE = 150   # UI Instagram
SIDE_SAFE = 100     # Кнопки Instagram справа

# Overlay
OVERLAY_WIDTH = 900
OVERLAY_X = (1080 - 900) // 2  # 90
OVERLAY_Y = 1400

# Шрифты
FONT_NAME_SIZE = 48
FONT_INFO_SIZE = 24
FONT_CTA_SIZE = 32
```

### Цвета UI

```python
UI_COLORS = {
    'overlay_bg': 'rgba(0, 0, 0, 0.75)',
    'text_primary': '#FFFFFF',
    'text_secondary': 'rgba(255, 255, 255, 0.8)',
    'accent': '#4A8F5C',  # цвет биома
    'border_radius': 16,
    'padding': 24,
}
```
